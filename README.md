# Password Strength Checker

## Overview
A Python-based cybersecurity project that evaluates password strength using multiple security checks.

## Features
- Password Length Validation
- Uppercase Letter Check
- Lowercase Letter Check
- Numeric Character Check
- Special Character Check
- Password Strength Scoring

## Technologies Used
- Python
- Regular Expressions (Regex)

## How It Works
The program analyzes a password and assigns a score based on:
1. Length >= 8 characters
2. Uppercase letters present
3. Lowercase letters present
4. Numbers present
5. Special characters present

## Example

Input:
Abc123@

Output:
Password Score: 5
Strong Password
